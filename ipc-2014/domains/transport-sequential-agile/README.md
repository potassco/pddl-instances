# Transport (Sequential, Agile)

## Domain Description

There is no description of this domain in the last IPC, but it seems to be a kind of logistics one.
Each vehicle can transport some packages depending on its capacity and moving has a cost depending on the length of the road.
Picking up or dropping a package costs 1.
